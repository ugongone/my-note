The "Eyes" for AI

Chrome DevTools MCP

AIはもう「コードを書くだけ」じゃない。
ブラウザを開き、クリックし、エラーを目視確認する時代へ。

何ができるのか？

自動操作 (Puppeteer)

「localhost:3000を開いてログインボタンを押して」と言えば、AIが実際にブラウザを操作して実行します。

コンソール監視

ブラウザのConsoleに出ているエラーログをAIが直接読み取り、バグの原因を即座に特定・修正案を出します。

視覚的確認

スクリーンショットを撮影したり、DOM構造を取得して「CSSでレイアウトが崩れている」ことを認識できます。

💻 フロントエンド開発の革命

コードを書いても、実際に動くかは別問題でした。このMCPを使えば、AIは以下のフローを完結できます。

  1. コードを修正する
  2. ブラウザでページをリロードする
  3. コンソールエラーが消えたか確認する
  4. まだエラーなら、ログを読んで再修正する

🧪 E2Eテストの自動生成

「ユーザー登録フローをテストして」と頼むだけで、AIが入力項目を埋めて送信テストを行い、その操作ログからPlaywright等のテストコードを生成することも可能です。

インストール (Claude Code)

以下のコマンドでプロジェクトに追加できます。

claude mcp add chrome-devtools npx chrome-devtools-mcp@latest
注意: 実行にはGoogle Chromeがインストールされている必要があります。ヘッドレスモード（画面なし）で起動させるオプションもあります。
