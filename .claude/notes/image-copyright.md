# 画像引用ポリシー（genai-news用）

## 基本方針

公式ブログ・プレスリリースの画像を引用する際は、以下の条件を確認して判断する。

## 引用の可否を決める4つの軸

| 軸 | OK寄り | NG寄り |
|---|---|---|
| 公開範囲 | 個人利用・限定共有（社内・知人） | 不特定多数への一般公開 |
| 営利性 | 非営利・学習目的 | 収益化あり |
| 画像の種類 | モデルのサンプル出力、図解、アーキテクチャ図 | ロゴ、製品写真、マーケティング素材 |
| 必然性 | 文章だけでは説明しにくい内容 | 文章で代替できる装飾的な用途 |

## 引用時の必須事項

- 出典を明記する：`© [企業名]、[ページタイトル] — [サービス名] より引用`
- 出典URLをリンクとして記載する
- 画像に `alt` テキストを付ける

## このアプリ（genai-news）の利用用途

genai-newsの記事に画像を掲載する場合、以下の条件が固定されている。

| 条件 | 内容 |
|---|---|
| 公開範囲 | 限定共有（社内・知人）。不特定多数への一般公開なし |
| 営利性 | 非営利・社内勉強用。収益化なし |
| コンテンツの性質 | ニュース記事の補足説明。画像はあくまで「従」 |
| 出典明記 | © 企業名・出典URLを記事内に必ず記載 |

**この用途での判断：** 上記4条件が揃っている限り、公式ブログ・プレスリリースのサンプル画像（モデル出力例・図解・アーキテクチャ図）の引用は、日本著作権法32条の引用要件を満たす可能性が高く、**OKと判断してよい。**

ただし、以下の場合は再判断が必要：
- 一般公開（Web公開）に切り替える場合
- 収益化に関連する用途になる場合
- 画像がプレスキットとして提供されていないロゴ・製品写真・マーケティング素材の場合
