なぜこの3つなのか？

これら3つのMCPサーバーは、開発者の三大要素である「コード管理 (GitHub)」、「データ操作 (Postgres)」、「外部知識の取得 (Brave Search)」を網羅しているため、Claude Codeの基本セットアップとして推奨されています。他にも、Google Drive（ドキュメント管理）や Chrome DevTools（ウェブテスト）も人気があります。

※MCP (Model Context Protocol) はオープン標準であり、使用するにはローカル環境へのセットアップが必要です。
