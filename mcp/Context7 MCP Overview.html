Docs for AI

Context7 MCP

「その機能、もう古いです」と言わせない。
AIに最新の公式ドキュメントを動的に注入する専門家。

Before

これまでの課題

  • ❌ 知識の期限切れ: 「そのライブラリのv2.0は私の知識（2024年まで）にはありません」と断られる。
  • ❌ 幻覚（ハルシネーション）: 存在しない関数や、廃止された古い書き方を自信満々で提案してくる。
With Context7

Context7の解決策

  • ✅ リアルタイム・ドキュメント: 15,000以上のライブラリの「今」のドキュメントを瞬時に取得。
  • ✅ 正確なコード生成: 公式のコード例をそのままプロンプトに混ぜるため、文法ミスが激減する。

主な機能

ドキュメント検索

`get-library-docs` ツールを提供。ユーザーがライブラリ名を指定するだけで、関連ページを自動で探して要約します。

コード例の注入

解説文だけでなく、実際の「Usage（使い方）」コードスニペットを優先的に取得するため、コピペで動くコードが生成されやすくなります。

高速・軽量

Web検索（Brave等）よりもデータ量が最適化されており、必要な情報だけをピンポイントで取得するためトークン効率が良いです。

Brave Search との違いは？

Brave Search MCP

「広範なウェブ検索」が得意。

  • ブログ記事、フォーラム(StackOverflow)の検索
  • トレンドや一般的な知識の調査
  • エラーメッセージの解決策探し

Context7 MCP

「一次情報の取得」に特化。

  • 公式ドキュメントのみを対象にする
  • APIの引数や戻り値の正確な定義
  • アップデート直後の最新ライブラリ仕様
